# Stack Overflow Developer Survey Analysis

### 🛠 Tools & Technologies
* **Language:** Python
* **Libraries:** Pandas
* **Environment:** Jupyter Notebook

### 📝 Project Description
Data-driven research of the global IT market based on the **Stack Overflow Developer Survey** (49,191 respondents). The project focuses on identifying trends in developer compensation, programming language popularity, and the evolution of remote work.

### 🔍 Key Tasks
* **Data Cleaning:** Handled missing values and filtered the dataset to ensure data integrity.
* **Statistical Analysis:** Calculated measures of central tendency (mean, median) and percentiles to analyze salary distributions.
* **Data Grouping:** Aggregated data by regions, education levels, and industries to extract meaningful insights.
* **Exploratory Data Analysis (EDA):** Investigated correlations between age groups and technology adoption.

### 📊 Key Results & Insights
* **Language Trends:** Python is used by **37.5%** of developers, with the highest adoption rate among the youngest demographic (18–24 age group).
* **Compensation Leaders:** The top 3 countries by median compensation for Python developers are:
  1. **USA** ($150K)
  2. **Switzerland** ($143K)
  3. **Israel** ($143K)
* **Remote Work:** Fintech, Healthcare, and Software Development industries offer the most competitive conditions for remote specialists.
